Luces Navideñas Animadas

Este proyecto consiste en una aplicación web que simula luces navideñas utilizando JavaScript y manipulación del DOM. El objetivo es generar dinámicamente todos los elementos en pantalla y animar las luces mediante patrones de colores que cambian de forma automática.

Descripción del Proyecto

La aplicación crea una fila de diez luces que cambian su color siguiendo una serie de patrones predefinidos. Toda la estructura visual (título, contenedor, luces y controles) se genera con JavaScript, mientras que la animación se logra con la función setTimeout, permitiendo que los patrones se ejecuten de manera cíclica e infinita.
Además, se agregaron funcionalidades adicionales como controles de velocidad, pausa, reanudación, modo oscuro y la opción de generar patrones aleatorios.

Tecnologías Utilizadas

-HTML5

-CSS3

-JavaScript (ES6+)

Instrucciones de Uso

-Clonar o descargar el repositorio.

-Abrir el archivo index.html en el navegador.

-La animación comienza automáticamente.

-Utilizar los botones para:

-Pausar la animación

-Reanudar la animación

-Aumentar la velocidad

-Disminuir la velocidad

-Generar un patrón aleatorio

-Activar el modo oscuro

Patrones de Colores

Los patrones predefinidos incluyen:

Patrón 1: rojo y verde alternados

Patrón 2: azul y amarillo alternados

Patrón 3: secuencia verde-verde / rojo-rojo

Patrón 4: secuencia amarillo, azul, rojo

Patrón 5: cinco luces rojas seguidas de cinco luces verdes

También es posible generar nuevos patrones aleatorios desde los controles.
